# Pub Crawl Landing Page

This is a lightweight static landing page for a pub crawl event.

## Edit before publishing

Update these values in `index.html`:

- Event date, time, location, and dress code
- The `Grab Tickets` and footer ticket URLs
- Event description, inclusions, and FAQ copy

## Free hosting options

- GitHub Pages
- Netlify
- Vercel

Because this site is just `index.html` and `styles.css`, all three can host it on a free plan.

## Local preview

Open `index.html` in a browser, or from this folder run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
